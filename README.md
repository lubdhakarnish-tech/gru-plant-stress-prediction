# GRU-Based Plant Stress Prediction

## 📌 Overview
This project uses a GRU (Gated Recurrent Unit) neural network to predict early-stage plant stress using time-series environmental data.

## ⚙️ Features
- Time-series modeling using GRU
- Multi-parameter input (temperature, humidity, soil moisture)
- Sliding window sequence generation
- Early prediction of plant stress

## 🧠 Model
- GRU-based recurrent neural network
- Trained on environmental sensor data
- Designed for temporal pattern learning

## 📊 Input Data
- Temperature
- Humidity
- Soil Moisture

## 🚀 Applications
- Smart agriculture
- Predictive irrigation systems
- Resource optimization

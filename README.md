# GRU-Based Plant Stress Prediction

## 📌 Overview
Predicts plant stress using GRU on time-series environmental data.

## ⚙️ Models
- GRU (deep learning)
- Random Forest (baseline comparison)

## 📊 Evaluation
- Accuracy, Precision, Recall
- ROC Curve
- Confusion Matrix

## 📈 Results
GRU outperformed Random Forest in capturing temporal dependencies.
